# FH6Macros

Два Windows-макроса в одном терминальном приложении. При запуске выбирается режим:

1. **Goliath** - постоянно удерживает `W` с редкими короткими отпусканиями (анти-AFK).
2. **Crush Combo** - удерживает `W` со случайными микро-отпусканиями и короткими тапами `S`, затем выполняет `X` -> `Enter` -> `Enter` и повторяет(Самостоятельно проходит трассу и перезапускает карту).

Управление из терминала и через хоткей `Ctrl + Up`.

## Использование

```powershell
go run .
```

Команды в терминале:

- `start` или `s` - запустить макрос;
- `stop` или `x` - остановить макрос;
- `edit` или `e` - изменить тайминги текущего режима;
- `show` - показать текущие настройки;
- `mode` или `m` - переключиться между Goliath и Crush Combo (макрос должен быть остановлен);
- `quit`, `q` или `exit` - выйти.

## Сборка

```powershell
go build -o FH6Macros.exe .
```

## Структура проекта

- `main.go` - меню, CLI-цикл, диспатч режимов
- `winapi.go` - привязки WinAPI, отправка клавиш, слушатель хоткея
- `common.go` - общее состояние, RNG, CLI-хелперы
- `goliath.go` - режим Goliath (удержание W)
- `crushcombo.go` - режим Crush Combo (W + S + X + Enter + Enter)
