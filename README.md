# FH6AutoW

Простой Windows-макрос для удержания клавиши `W` с периодическим коротким отпусканием. Управление доступно из терминала и через хоткей `Ctrl + Up`.

## Использование

```powershell
go run .
```

Команды в терминале:

- `start` или `s` - запустить макрос;
- `stop` или `x` - остановить макрос;
- `edit` или `e` - изменить тайминги;
- `show` - показать текущие настройки;
- `quit`, `q` или `exit` - выйти.

## Сборка

```powershell
go build -o FH6AutoW.exe .
```
